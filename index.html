Help & FAQ

What is a Telegram Session?

A session allows your application to authenticate with Telegram without entering credentials repeatedly. It's like a saved login for bots and userbot applications.

How to get API Credentials?

  1. 1. Go to my.telegram.org
  2. 2. Login with your phone number
  3. 3. Click "API Development Tools"
  4. 4. Fill the form to create a new application
  5. 5. Copy your API ID and API Hash

Telethon vs Pyrogram?

Telethon: Mature library with extensive documentation, great for complex bots.
Pyrogram: Modern library with cleaner API, easier for beginners.

Is this safe?

Yes! This website runs entirely in your browser. No data is sent to external servers. Your credentials and sessions remain private.

Common Issues

  • • Invalid credentials: Double-check your API ID and Hash
  • • Phone format: Use international format (+1234567890)
  • • Code not received: Check your Telegram app, not SMS
  • • 2FA required: Enter your cloud password, not phone passcode
